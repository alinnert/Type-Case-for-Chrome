Type Case for Google Chrome

A text editor for Chrome OS. It's the equivalent to Windows Notepad plus some neat extras.

Open in Chrome Web Store

A text editor for Chrome OS. It's the equivalent to Windows Notepad plus some neat extras.

Currently I'm working on something bigger, so this needed to be a "quick'n'dirty" solution because I just missed an app like this. So functionality is very basic for now. Take care when you close the app!

KEYBOARD SHORTCUTS

  • ctrl + n = create a new file
  • ctrl + s = save current file
  • ctrl + shift + s = save as new file
  • ctrl + o = open file
  • ctrl + komma = open settings
  • ctrl + + = increase font size
  • ctrl + - = decrease font size
